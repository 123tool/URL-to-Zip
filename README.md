## URL TO ZIP All-in-One

Tool profesional untuk mengunduh source code website secara lengkap (HTML, CSS, JS, Images) dalam satu paket ZIP.

## 🚀 Fitur
- **Auto Assets Downloader**: Mendownload gambar, file CSS, dan JS secara otomatis.
- **Path Remapping**: Link aset di HTML otomatis dirubah ke lokal agar bisa dibuka offline.
- **Dual Interface**: Tersedia versi Web GUI (Localhost) dan CLI (Terminal).
- **Auto ZIP**: Hasil kloning otomatis dikompres menjadi file ZIP/RAR.

## 🛠️ Instalasi

### 1. Prasyarat
Pastikan sudah terinstall **Python 3.x**.

### 2. Clone
```
git clone https://github.com/123tool/URL-to-Zip.git
cd URL-to-Zip
```
### 2. Install Library
Jalankan perintah berikut di Terminal/CMD/Termux:
```bash
pip install -r requirements.txt
```

## Cara Menjalankan

​Opsi A :
Menggunakan Web GUI (Browser) ​Jalankan server :
```
python app.py
```
Buka browser dan akses : 
```
http://localhost:5000
```
​Opsi B :
Menggunakan CLI (Terminal) ​Jalankan script :
```
python main_cli.py
```
